## Learning Goal

After finishing these notes, you should be able to:

- write and read lambda expressions confidently
- design your own functional interfaces
- use lambdas with Collections and Streams fluently
- choose between lambda, method reference, and normal class
- handle real-world cases (exceptions, performance, reuse, testing)
